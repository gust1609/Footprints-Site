Gabon Scorekeeper

Privacy Policy

Effective date: June 7, 2026

Overview

Gabon is a scorekeeper for a physical card game. It stores only the information needed to create players, save games, calculate score history, and keep the app useful on your device.

Data Stored On Your Device

  • Player names, initials, profile colors, optional profile photos, and local profile settings.
  • Game titles, participating players, round scores, correction events, halving events, final scoreboards, and statistics.
  • Optional game location when you request a location-based title while creating a game.

If location permission is denied or a GPS fix cannot be obtained, no coordinates are stored.

How Data Is Used

The app uses this data to remember your active player profile, save game history, show score statistics, and keep active games available locally.

Tracking and Advertising

Gabon does not show third-party ads, does not sell data, and does not use collected data for tracking.

Cloud Sync

The submitted App Store build does not include a cloud configuration file and runs fully offline with local storage only. It does not send player, score, photo, or game-location data to a developer-operated cloud backend.

If a future build enables cloud sync, this policy will be updated before that release to describe the synced data and service providers.

Location and Photos

Location permission is optional and requested only when creating a new game title. Optional photo access is used only when you add a player profile picture.

Delete or Correct Data

You can edit player profiles and delete games inside the app. For privacy requests, email gustav.manthorpe@gmail.com.
