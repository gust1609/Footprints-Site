# Footprints Site

Den offentlige, danske koncept- og supportside for Footprints.

## Lokal visning

Kør en statisk server fra projektmappen:

```bash
python3 -m http.server 4173
```

Åbn derefter `http://127.0.0.1:4173/`.

Den iterative design- og QA-proces er beskrevet i [`ITERATION_PROMPT.md`](ITERATION_PROMPT.md).
